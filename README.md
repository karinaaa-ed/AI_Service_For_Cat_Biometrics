# Cat Iris Biometrics
ИИ-сервис для биометрии кошек на основе анализа радужки глаза 

(AI Service For Cat Biometrics Based On Iris Analysis)

## Описание проекта
Данный проект посвящён разработке ИИ-сервиса для биометрического анализа кошек на основе изображений глаз, в частности — радужки глаза. Конечной целью проекта является исследование возможности определения пола и возраста кошки по изображению радужной оболочки с использованием методов компьютерного зрения и машинного обучения.
Основная идея заключается в том, что рисунок радужной оболочки глаза может использоваться как биометрический признак для идентификации животных, аналогично биометрии человека.

**Проект находится в активной стадии разработки и будет дополняться новыми этапами по мере продвижения.**

## Цели и задачи

**Глобальная цель**

Разработать ИИ-сервис, способный:
- анализировать изображения глаз кошек,
- извлекать область радужки,
- определять пол и возраст кошки по биометрическим признакам радужки.
