# Cat Iris Biometrics
ИИ-сервис для биометрии кошек на основе анализа радужки глаза 

(AI Service For Cat Biometrics Based On Iris Analysis)

## Описание проекта
Данный проект посвящён разработке ИИ-сервиса для биометрического анализа кошек на основе изображений глаз, в частности — радужки глаза. Конечной целью проекта является исследование возможности определения пола и возраста кошки по изображению радужной оболочки с использованием методов компьютерного зрения и машинного обучения.
Основная идея заключается в том, что рисунок радужной оболочки глаза может использоваться как биометрический признак для идентификации животных, аналогично биометрии человека.

**Проект находится в активной стадии разработки и будет дополняться новыми этапами по мере продвижения.**

## Цели и задачи

**Глобальная цель**

Разработать ИИ-сервис, способный:
- анализировать изображения глаз кошек,
- извлекать область радужки,
- определять пол и возраст кошки по биометрическим признакам радужки.

**Задачи текущего этапа**
- Изучить существующие научные работы по биометрии животных и анализу радужки.
- Подобрать и проанализировать подходящий датасет.
- Реализовать детекцию глаз кошек на изображениях лиц.
- Построить и сравнить baseline-модели для задачи детекции.

## Обзор литературы

На текущем этапе был проведён обзор научных публикаций по следующим направлениям:
- биометрическая идентификация животных;
- компьютерное зрение в ветеринарии;
- анализ радужки глаза (iris recognition);
- детекция ключевых точек и объектов на изображениях животных.

Результаты обзора используются для формирования дальнейшей методологии и выбора архитектур моделей.

## Данные

В качестве исходного датасета используется:

[CatFLW — Cat Facial Landmarks in the Wild](https://github.com/martvelge/CatFLW) 

### Описание датасета
- **2079 изображений** кошачьих лиц в реальных условиях.
- **48 лицевых ориентиров (facial landmarks)** на каждом изображении.
- **Bounding box** для лица кошки.
- Разнообразие пород, условий освещения и ракурсов.

Датасет изначально не содержит аннотаций радужки, поэтому на текущем этапе он используется для:
- детекции лица и глаз;
- подготовки данных для последующей разметки области радужки.

## Модели и методы

### Детекция глаз кошек (Baseline)

На текущем этапе реализованы baseline-модели для детекции глаз на изображениях кошачьих лиц:
- YOLOv5
- YOLOv11

Модели обучались на подготовленных аннотациях, полученных на основе landmark-разметки датасета CatFLW.
